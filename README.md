# movie_app


A simple interest movie app which provide real-time upcoming movies .



## Implementations
- Bloc Architecture
- Themoviedb.org Api
- Dio
- Carousel slider
- Url launcher
- Hexcolor
- Cached_network_image



## Features
- Has both Dark & Light Mode
- Clean & Easy Navigation
- Simple and fast 
- Youtube trailer
- Local storage
- Pagination
